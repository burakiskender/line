Line Server Class Documentation is here

new Server(options: [Object])

Extends EventEmitterExtra

Parameters
options ([Object]) Optional options object.
Name Description
options.host [string] Server host name. Inherited from uws, see docs .
options.port [number] Server port. Inherited from uws, see docs .
options.server [http.Server] Server object to be attached. If provided, host and port will ignored. Inherited from uws, see docs .
options.handleProtocols [Function] Inherited from uws, see docs .
options.path [string] Inherited from uws, see docs .
options.noServer [boolean] Inherited from uws, see docs .
options.clientTracking [boolean] Inherited from uws, see docs .
options.perMessageDeflate [(boolean | Object)] Inherited from uws, see docs .
options.timeout [number] Timeout duration (in ms) for message responses. Default: 30 seconds
options.maxReconnectDelay [number] Maximum reconnection delay (in seconds) for clients. Default: 60 seconds
options.initialReconnectDelay [number] Intial reconnection delay (in seconds) for clients. Defualt: 1 seconds
options.reconnectIncrementFactor [number] Reconnection incremental factor for clients. Default: 2
options.pingInterval [number] Ping interval (in ms) for both server and client. Default: 60 seconds.
Example
const Server = require('line/server');
const server = new Server({
    port: 8080
});

Server is an full-fledged event emitter.

Events
Properties
Server.Events.CONNECTION ("connection")
Server.Events.HANDSHAKE ("handshake")
Server.Events.HEADERS ("headers")
Server.Events.ERROR ("error")
Example
// On a client connects after successful handshake
server.on('connection', (connection) => {
   connection.send('hello');
   ...
});

// If you want to authorize your client
server.on('handshake', (connection, handshake) => {
   if (handshake.payload.token == 'test')
       handshake.resolve();
   else
       handshake.reject(new Error('Invalid token'));
});

// Catch errors
server.on('error', (err) => {
   ...
});
